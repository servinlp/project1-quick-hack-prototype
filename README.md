# project-oba
Using the data from the OBA to create a catelog

### Requirements

- Yarn
- Sass
- php

### To run the project

```
yarn run start
```

This will start a php localhost server at [http://localhost:4000] and will run sass watch.

[http://localhost:4000]: http://localhost:4000
